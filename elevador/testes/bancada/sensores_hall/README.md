# Teste — Sensores Hall

Validação dos sensores de piso tipo Hall antes da integração na maquete.

| Ficheiro | Descrição |
|----------|-----------|
| [sensorhall.ino](sensorhall.ino) | Leitura digital de sensores Hall |

## Lógica

- `LOW` = sensor activo (ímã / cabine presente)
- Substituem os interruptores da simulação Wokwi

## Pinagem prevista (maquete)

| Sensor | Pino |
|--------|------|
| Piso 1 | D8 |
| Piso 2 | D11 |
| Piso 3 | D12 |
| Piso 4 | D13 |
| Porta | D22 |

## Próximo passo

Montar na maquete (E08) e validar com firmware real.
