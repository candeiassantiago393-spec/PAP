```text
simulacao/
├── legacy/
│   ├── tinkercad/     → v01, v02 (Arduino Uno, motor DC)
│   └── wokwi/         → v03–v06 (Arduino Mega, A4988, OLED)
└── wokwi/
    └── exports/       → diagram.json + wokwi.toml (pendente)
```
