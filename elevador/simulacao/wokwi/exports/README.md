# Wokwi — Exports adicionais

Pasta reservada para cópias extra de projetos Wokwi (backups, variantes, exports futuros).

## Estado

**Vazia por intenção.** Os exports activos estão em:

`elevador/simulacao/legacy/wokwi/v03–v06/wokwi/`

## Quando usar esta pasta

- Exportar uma nova versão v07+ antes de a integrar em `legacy/`
- Guardar snapshot antes de alterações grandes
- Duplicar um projeto para experimentação sem tocar na versão de referência

## Como exportar do Wokwi

1. Abrir projeto em https://wokwi.com
2. Menu → Download / exportar ficheiros
3. Colocar aqui numa subpasta, ex.: `exports/v07_experimento/wokwi/`
