# Wokwi — Simulação

Exports e cópias de projetos Wokwi.

## Onde estão os ficheiros Wokwi

Os projetos **v03–v06** estão integrados em:

```text
simulacao/legacy/wokwi/
├── v03_transicao_mega_a4988/wokwi/
├── v04_7segmentos_reset_rearme/wokwi/
├── v05_oled_tca9548a/wokwi/
└── v06_atual_pedidos_internos_porta/wokwi/   ← versão de apresentação
```

Cada pasta `wokwi/` contém: `diagram.json`, `sketch.ino`, `wokwi.toml`, `libraries.txt`.

## Pasta `exports/`

Reserva para **cópias adicionais** ou novos exports. Por agora vazia — ver [exports/README.md](exports/README.md).

## Simular no Cursor

1. Abrir pasta `.../v06_.../wokwi/`
2. `F1` → **Wokwi: Start Simulator**
